## 📌 Project Overview
This project focuses on developing essential digital literacy skills required in today’s digital world. It aims to promote responsible use of digital tools, effective online communication, and awareness about cyber safety.  

The project consists of five tasks that cover infographic creation, digital portfolio development, platform usage, email etiquette, and cybercrime awareness. It helped in building practical knowledge and improving digital skills.
